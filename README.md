# PerpusKu — Sistem Perpustakaan

Aplikasi manajemen perpustakaan berbasis PHP dan SQLite untuk keperluan pembelajaran pengujian perangkat lunak.

## Fitur
- Pencarian dan daftar buku
- Registrasi & login anggota
- Peminjaman buku
- Pengembalian buku dengan perhitungan denda keterlambatan (Rp 500/hari)
- Kelola buku dan peminjaman (admin)

## Cara Menjalankan

1. **Clone atau unduh** folder `php-perpustakaan`.
2. **Jalankan server PHP:**
   ```bash
   cd php-perpustakaan
   php -S localhost:8003
   ```
3. **Buka browser** dan akses `http://localhost:8003`
